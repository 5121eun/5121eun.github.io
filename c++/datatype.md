# Data type

| Datatype       | size         | 
| --------------- | -------------- |
| boolean          | 1 byte  |
| char          | 1 byte  |
| int         | 2 or 4 byte  |
| float         | 4 byte  |
| double         | 8 byte  |

| Datatype        | example         | 
| --------------- | -------------- |
| arrays          | int numbers[]  |
| reference       | int &no = num  |
| pointer         | int* ptr = &num  |
