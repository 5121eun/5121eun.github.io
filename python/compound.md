# Compound

## If

```python
if False:
    pass
elif False:
    pass
else:
    pass
```

## Match
new in 3.10

```python
match 5 % 2:
    case 0:
        print("5 is even")
    case 1:
        print("5 is odd")
```

## while

```python
count = 0
while count < 5:
    count += 1
    print(count)
else:
    print("end")
```

## for


```python
for i in range(5):
    print(i)
else:
    print("end")
```

## try

```python
try:
    1/0
except:
    print("error")
else:
    print("no error")
finally:
    print("end")
```
